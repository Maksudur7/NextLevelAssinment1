1.What are some differences between interfaces and types in TypeScript?
Ans : যদিও ইন্টারফেস এবং টাইপ প্রায় একই কাজ করে তবে এদের ভিতরে পার্থক্যটা হলো declaration এ।  ইন্টারফেসকে একই স্কোপে একাধিকবার ঘোষণা করা যেতে পারে। এবং পরে সেগুলো একত্রে করে করা যায়। কিন্তু টাইপের ক্ষেত্রে একাধিকবার ঘোষণা করা যায় না, কিন্তু টাইপ ব্যবহার করে জটিল ধরনের টাইপ গুলো হ্যান্ডেল করা যায়। যেমন শর্ত যুক্ত টাইপ ম্যাপিং টাইপ ইত্যাদি। ইন্টারফেস এই জটিল ধরনের টাইপগুলো সরাসরি হ্যান্ডেল করতে পারে না এটা ইন্টারফেস এর সীমাবদ্ধতা।

2.What is the use of the keyof keyword in TypeScript? Provide an example.
Ans : keyof কিওয়ার্ডে শহ হল টাইপ নিরাপত্তা নিশ্চিত করা।  এটি আমাদের এমন ফাংশন তৈরি করতে সাহায্য করে যা শুধুমাত্র একটি নির্দিষ্ট অবজেক্ট এর বৈধ প্রপার্টির নাম গুলো ইমপোর্ট হিসেবে গ্রহণ করবে। যেমন আমি যদি কোন ফাংশনে একটি ডায়নামিক অবজেক্ট পাঠাতে চাই,  তখন আমার কীবোর্ডের টাইপ গুলো ভিন্ন ভিন্ন হতে পারে অনেক রেনডোমলি হতে পারে। এর জন্য ওই অবজেক্টকে যদি টাইপ অফ দিয়ে ওই কি গুলোকে টাইপ হিসেবে  ইউজ করা যায় এর জন্য keyof keyword  ব্যবহার করা হয় l
 Example :  

`type RichPeoplesVehicle = {
    car : string;
    bike : string;
    cng : string;
}
type MyVehicle2= keyof RichPeoplesVehicle
const myVehicle : MyVehicle2 = 'bike'`
